# stream-notify-room-users

This is the stream for room users. Currently used to notify about WebRTC Calls.

Events available:

* webrtc

```javascript
{
    "msg": "sub",
    "id": "unique-id",
    "name": "stream-notify-room-users",
    "params":[
        "user-id/event",
        false
    ]
}
```
