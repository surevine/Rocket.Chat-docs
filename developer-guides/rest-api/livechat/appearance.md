# Livechat appearance data

Get Livechat widget settings about appearance

| URL | Requires Auth | HTTP Method |
| :--- | :--- | :--- |
| `/api/v1/livechat/appearance` | `yes` | `GET` |

## Example Call

```bash
curl http://localhost:3000/api/v1/livechat/appearance
```
