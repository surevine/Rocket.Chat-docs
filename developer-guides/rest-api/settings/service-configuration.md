# Service Configurations Get

List all service configurations.

| URL | Requires Auth | HTTP Method |
| :--- | :--- | :--- |
| `/api/v1/service.configurations` | `no` | `GET` |

## Example Call

```bash
curl http://localhost:3000/api/v1/service.configurations
```

## Example Result

```javascript
{
  "configurations":  [],
  "success": true
}
```
